Instructions:

1. Ensure you have Maple 2024 (or newer) installed on your system.

2. Open a command window using Git Bash.

3. Navigate to the folder containing the solver file:
   Restart_Maple_XX_BDFX_V.txt

4. Use the following command to run the solver using Maple in command-line mode:

-------------------------------------
/c/Program\ Files/Maple\ 2024/bin.X86_64_WINDOWS/cmaple.exe < Restart_Maple_XX_BDFX_V.txt
-------------------------------------

Note: This command must be executed from the **same folder** where the `.txt` file is located.
